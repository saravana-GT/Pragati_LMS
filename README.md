# Pragati Abhyasika Dashboard 🚀

A premium, modern Learning Management System (LMS) administrative panel designed for study room and library management. Built with **React**, **Vite**, and **Firebase**, it provides a seamless user experience with a focus on ease of use and professional aesthetics.

## ✨ Key Features

- **📊 Advanced Analytics:** Modern dashboard widgets showing active students, occupied seats, and revenue status at a glance.
- **🪑 Visual Seat Mapping:** Interactive seat layout with shift-based occupancy tracking (Morning/Evening shifts).
- **👥 Student Management:** Easily add, list, search, and manage students with a responsive UI.
- **💰 Fee & Payment Tracking:** Automated generation of monthly payment records and status tracking.
- **✅ Attendance System:** Simple check-in/check-out management for daily operations.
- **🔐 Secure Authentication:** Built with Firebase Auth for robust security.
- **🎨 Premium UI/UX:** Clean, responsive design with dark/light mode accents and smooth micro-animations.

## 🛠️ Built With

- **Frontend:** React 18, Vite
- **Styling:** Vanilla CSS (Custom Utility System)
- **Backend/Database:** Firebase Firestore
- **Authentication:** Firebase Auth
- **Routing:** React Router (HashRouter for GH Pages compatibility)
- **Toasts:** React Hot Toast
